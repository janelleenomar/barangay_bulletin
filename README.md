### Project Description
Barangay Bulletin is an offline-first mobile application that serves as a digital community board for a barangay. Residents can post and browse announcements, log infrastructure or community issues, track the status of those reports, and manage archived (soft-deleted) entries — all without requiring internet access. 

This project demonstrates core Flutter development concepts, including structured nested navigation with strict data passing, local database persistence via Hive, and stateful UI management relying exclusively on Flutter's built-in `setState`.
